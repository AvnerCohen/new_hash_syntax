To_New_Hash
--
Ruby utility script to replace pre 1.9 hash syntax to post 1.9 syntax (:var => to var:)

Run test:

    rspec test.rb

Run script:

run.rb [filename|directory] [FULL]
[FULL] -- Include code, not only remarks
